# Customer Segmentation using K-Means Clustering

## Project Overview
This project segments customers into different groups using machine learning based on:

- Age
- Gender
- Annual Income
- Spending Score
- Purchase Frequency

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit

## Features
- Data preprocessing
- Feature engineering
- K-Means clustering
- Elbow method
- Cluster visualization
- Business recommendations
- Interactive dashboard

## Files Included
- customers.csv
- Customer_Segmentation.ipynb
- customer_segmentation_output.csv
- app.py

## Output
The system groups customers into:

- Premium Customers
- Budget Active Customers
- Regular Customers
- Low Value Customers
